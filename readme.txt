Start to monitor your blog's uptime with www.website-monitoring.com services - and have all the charts and tables displayed in your WordPress panel.

== Description ==

This plugin integrates the user panel of website-monitoring.com with WordPress administration panel.
After installation and giving the proper login and password, the uptime history (tables and charts) as well as service configurations are available in WordPress without the need to login separately.

[Read more about website monitoring services](http://www.website-monitoring.com/?ps=4 "website availability monitoring")

== Installation ==

1. Upload the whole directory "website-monitoring" to your blog's plugin directory (usually "/wp-content/plugins").
2. Login to your WordPress panel, go to "Plugins" section and mark the new plugin active.
3. Give your website-monitoring.com login details (login and password) in "Settings > Website Monitoring".
4. Done.
