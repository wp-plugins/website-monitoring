Monitor your blog's uptime with www.supermonitoring.com services - and have all the charts and tables displayed in your WordPress panel.

== Description ==

This plugin integrates the user panel of supermonitoring.com with WordPress administration panel.
After installation and configuration (entering an authorization token), the uptime history (tables and charts) as well as service configurations are available in WordPress without the need to login separately.

[Read more about website monitoring services](http://www.supermonitoring.com/ "website monitoring")

== Installation ==

1. Upload the entire directory "supermonitoring" to your blog's plugin directory (usually "/wp-content/plugins").
2. Login to your WordPress panel, go to "Plugins" section and mark the new plugin active.
3. Log in yo your supermonitoring.com account and obtain an authorization token in "Account Settings" section.
4. Paste the token into plugin configuration in your WordPress Panel: "Settings > Super Monitoring".
5. Done.
